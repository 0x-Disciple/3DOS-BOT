### 3DOS-BOT

## Features
- Auto ping
- Support multi account
- Error handling

## Installation
1. Clone this repository
```bash
git clone https://github.com/0x-Disciple/3DOS-BOT.git cd 3DOS-BOT
```
2. Install dependencies
```bash
npm install
```
3. Fill in your secretKey & bearer in `data.txt`

4. Create screen session
```bash
screen -S 3DOS
```
5. Run 
```bash
node index.js
```
